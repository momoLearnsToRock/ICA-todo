This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

item created

get /todo
searches for todos (searchTodo)
By passing in the appropriate options, you can search for available todos in the system

Query parameters

id (optional)
Query Parameter — fetch a specific ToDo
assignedTo (optional)
Query Parameter — fetch ToDos for a specific user or group
category (optional)
Query Parameter — fetch ToDos with a specific category
skip (optional)
Query Parameter — number of records to skip for pagination format: int32
limit (optional)
Query Parameter — maximum number of records to return format: int32

Return type
